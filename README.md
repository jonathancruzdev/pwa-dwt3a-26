# GIT
1. Iniciamos un repositorio de GIT  git init
2. Configuramos el usuario
    - git config user.name "Jonathan Cruz"
    - git config user.email jonathan.cruz@ 
3. Agregamos los archivos al repo   git add README.md
4. Hacemos el commit (Foto)    git commit -m "Proyecto inicial"
5. Vinculamos el repositorio remoto git remote add origin https://github.com/user
6. Realizamos el push git push origin master

git add .
